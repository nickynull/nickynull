# 使用 VuePress 实例

命令

npm install     # 安装
npm run dev     # 运行
npm run build   # 打包

使用建议： VSCode + Nodejs

# 关于 VuePress 实例

  • 已经配置 VuePress 基础设置与样式
  • 使用者只需关注内容创作
    • 在 docs 下编写 markdown 文档
    • 在 docs/config.js 中配置导航

使用实例前，所需具备的基础知识

  • VuePress：基础框架
  • Markdown：编写内容文档
  • Nodejs (npm)：运行平台/打包工具
