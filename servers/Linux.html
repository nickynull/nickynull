# Linux

Linux 是一种自由和开放源码的类 UNIX 操作系统

# 系统管理

# 常用通用参数

-y      # 自动确认
sudo    # 以root权限执行命令

# 查看系统信息

cat /proc/cpuinfo   # 查看 cpu 信息
cat /proc/meminfo   # 查看内存信息
uname -r            # 查看内核版本

# yum

基于rpm，软件包管理工具

# yum常用命令

sudo yum install 软件包名    # 安装
sudo yum remove 软件包名     # 卸载

# 常用软件

sudo yum install -y wget    # 下载软件
sudo yum install -y vim     # 增强版 vi
